Mythic Upgrades Wiki

Ores

Hover over an ore to see where it spawns. Each gem requires finding its biome in the correct dimension.

Blocks

All placeable blocks added by Mythic Upgrades, organised by gem. Block names match the in-game lang file.

Armor & Tools

Each gem provides a full set of tools and armor. Armor pieces unlock an ability whose power scales with how many pieces of that set you wear.

Status Effects

Custom effects applied by Mythic Upgrades armor and tools. All numeric values are sourced from MythicStats.java.
